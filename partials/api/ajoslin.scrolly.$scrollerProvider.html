$scrollerProvider (service in module ajoslin.scrolly )

Description

Used for configuring scroll options.

Methods

  • bounceBackDistanceMulti(newDistanceMulti)

    When the user scrolls past the content area into the bounce buffer, we need to bounce back. To decide how long the bounce back animation will take, there are two factors: a minimum time, in milliseconds, and a distance multiplier.

    The equation for deciding how much time the animation to bounce back to the main content area should take, we do the following:

    function getBounceTime(distancePastContent) {
  return bounceBackMinTime + distancePastContent * bounceBackDistanceMulti;
}

    This makes it so the farther away the user has scrolled from the content area, the longer the animation to bring the content back into view will take. The minimum time exists so even short distances still take a little bit of time.

    Parameters

    • newDistanceMulti(optional) – {number=} –

      The new bounce back distance multiplier.

    Returns

    {number} –

    bounceBackDistanceMulti The current bounce back distance multiplier.

  • bounceBackMinTime(newTime)

    See bounceBackDistanceMulti.

    Parameters

    • newTime(optional) – {number=} –

      The new bounce back minimum time to set.

    Returns

    {number} –

    bounceBackMinTime The current bounce back minimum time.

  • bounceBuffer(newBounceBuffer)

    Sets/gets the buffer allowed for the scroll to 'bounce' past the actual content area. Set this to 0 to effectively disable bouncing.

    Parameters

    • newBounceBuffer(optional) – {number=} –

      The new bounce buffer to set.

    Returns

    {number} –

    bounceBuffer The current bounce buffer.

  • pastBoundaryScrollRate(newRate)

    Sets/gets the rate scrolling should go when the user goes past the boundary. For example, if the user is at the top of the list and tries to scroll up some more, he will only be able to scroll at half the rate.
    Change this option to change 'half' to something else.

    Parameters

    • newRate(optional) – {number=} –

      The new pastBoundaryScrollRate to set.

    Returns

    {number} –

    pastBoundaryScrollRate The current scroll rate.
